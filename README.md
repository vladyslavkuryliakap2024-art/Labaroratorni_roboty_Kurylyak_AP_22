# Збірник Лабораторних Робіт з Програмування на Java

Цей репозиторій містить комплекс практичних та лабораторних робіт, виконаних на мові Java. Проєкти охоплюють шлях від базового синтаксису, керуючих конструкцій та масивів до складніших концепцій: об'єктно-орієнтованого програмування (ООП), колекцій (Collections Framework), файлового введення-виведення (I/O) та розробки графічних інтерфейсів (GUI) за допомогою бібліотеки Swing.

### 1. Основи Java та Демонстрація Об'єктів
Програма демонструє базову роботу з пам'яттю, об'єднання рядків та базове створення масивів об'єктів.
* **Функціонал:** Розрахунок суми 10 чисел; об'єднання масиву слів у речення; обчислення сумарного віку користувачів (клас `User`) та загальної потужності автопарку (клас `Car`); математичний алгоритм порозрядного розкладання тризначного числа для його дзеркального відображення (реверсу).

### 2. Інтерактивні Меню та Логічні Розгалуження
Програма побудована у вигляді інтерактивного консольного меню, де користувач обирає одне з завдань за допомогою `java.util.Scanner`.
* **Функціонал:**
  1. *Квадратне рівняння:* Розрахунок дискримінанту та пошук коренів рівняння $ax^2 + bx + c = 0$.
  2. *Квадрант:* Визначення координатної чмерті для точки $(x, y)$.
  3. *Перевірка числа:* Перевірка, чи є введене число двозначним та парним.
  4. *Шматкова функція:* Обчислення $f(x)$ залежно від діапазону значень $x$.
  5. *День тижня:* Використання покращеного `switch-case` для визначення типу дня (робочий/вихідний).

### 3. Цикли та Табулювання Функцій
Демонстрація еквівалентності та відмінностей між циклічними конструкціями, а також математичні обчислення.
* **Функціонал:** Повторення тестової фрази 50 разів; симуляція годинника (годин та хвилин) через вкладені `for` та `while`; алгоритм синхронізації часу за допомогою операцій ділення та залишку; табулювання функції $y = \sqrt{x} \cdot \cos(x)$ на проміжку $x \in [0; 2.05]$ з підрахунком кількості значень, що потрапили у заданий користувачем діапазон $[A, B]$.

### 4. Одновимірні Масиви та Ітерація
Практична робота зі структурами даних фіксованої довжини.
* **Функціонал:** Динамічне заповнення масивів парними та непарними числами за формулою; обхід масивів у прямому та зворотному напрямках; виведення елементів, що стоять лише на парних або непарних індексах; підрахунок суми елементів типу `double` через `for-each`; алгоритм інверсії (зміни знаку) для всіх непарних за значенням елементів масиву.

### 5. Декомпозиція Коду та Функції
Перехід від лінійної структури проєкту до створення перевикористовуваних статичних методів із чіткими сигнатурами.
* **Функціонал:** Метод `getArea` для розрахунку площі квадрата; метод `getMin` для порівняння трьох чисел; методи `printArr` та `getMax` для виведення та пошуку максимального значення у динамічно створеному масиві, розмір та вміст якого користувач задає самостійно під час виконання.

### 6. Фундаментальні Принципи ООП
Практична реалізація ключових концепцій об'єктно-орієнтованого програмування.
* **Функціонал:** 
  * *Інкапсуляція:* Створення об'єктів `Person` та `Phone` з конструкторами. Демонстрація виклику одного конструктора з іншого через `this(...)`.
  * *Поліморфізм:* Перевантаження методів `receiveCall` (з різною кількістю параметрів). Використання аргументів змінної довжини `Varargs` (`String... numbers`) для розсилки повідомлень.
  * *Абстракція та Успадкування:* Абстрактний клас `Car` задає обов'язковий шаблон, а класи-нащадки `Sedan` та `Truck` перевизначають (`@Override`) абстрактний метод `brake()` кожен у свій власний спосіб.

### 7. Java Collections Framework
Робота з динамічними структурами даних змінної довжини.
* **Функціонал:**
  * *Списки (`ArrayList<Car>`):* Створення динамічного автопарку, де кожен об'єкт `Car` містить посилання на об'єкт `Driver`. Реалізація циклів масової модифікації (ремонт мотору з підвищенням потужності, зміна водіїв, індексація цін, автоматичне коригування стажу водіїв за умовами). Фільтрація за допомогою `Random` для імітації виклику таксі.
  * *Хеш-карти (`HashMap<String, String>`):* Реалізація телефонної книги. Демонстрація CRUD-операцій: додавання (`put`), пошук за ключем (`get`, `containsKey`), пошук за значенням (`containsValue`), видалення (`remove`) та ітерація по парах ключ-значення через `Map.Entry`.

### 8. Файлове Введення-Виведення та Генерація Чисел
Взаємодія програми з дисковим простором комп'ютера та обробка виключних ситуацій.
* **Функціонал:** Інтерактивний запит імені файлу та параметрів. Перевірка наявності файлу через `file.exists()` та його створення. Потоковий запис згенерованих випадкових чисел у діапазоні $[b_1, b_2]$ за допомогою `FileWriter`. Посимвольне читання даних з файлу назад у консоль за допомогою `FileReader`. Увесь код обгорнуто в блок `try-catch` для захисту від помилок введення-виведення (`IOException`).

### 9. Графічний Інтерфейс Користувача (GUI Swing)
Створення десктопного віконного додатку з власною подієвою моделлю.
* **Функціонал:** Унаслідуваний від `JFrame` клас ініціалізує вікно із вимкненим менеджером компонування (`setLayout(null)`). Елементи розміщені за абсолютними координатами. Користувач вводить масу ($m$) та швидкість ($v$), натискає кнопку `JButton`, після чого анонімний клас-слухач подій обчислює кінетичну енергію за формулою:
  $$W_k = \frac{m \cdot v^2}{2}$$
  Поле результату захищене від редагування. Реалізовано перехоплення помилок формату чисел (`NumberFormatException`) із виведенням повідомлення про некоректне введення.
