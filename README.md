# Labaroratorni_roboty_Kurylyak_AP_22
# Лабораторна №1

Цей проєкт є консольною програмою на мові Java, яка демонструє базові принципи роботи зі змінними, текстовими рядками, масивами об'єктів та простими математичними алгоритмами.

## 📌 Функціонал програми

Проєкт складається з 5 основних логічних блоків, які виконуються послідовно у методі `main`:

1. **Арифметика змінних:** Розрахунок та виведення суми 10 цілих чисел.
2. **Робота з рядками:** Динамічне об'єднання масиву слів у повноцінне речення за допомогою `String.join()`.
3. **Облік користувачів:** Створення масиву об'єктів класу `User` та обчислення їхнього сумарного віку.
4. **Автопарк:** Створення масиву об'єктів класу `Car` (спортивні, класичні та електромобілі) з розрахунком їхньої загальної потужності в кінських силах.
5. **Математичний реверс:** Алгоритм розкладання тризначного числа на розряди та виведення його у зворотному порядку (дзеркально).

---

## 🛠 Структура класів

Проєкт містить три основні класи:

### 1. `User`
Зберігає персональні дані користувачів.
* Поля: `id`, `age` (вік), `name` (ім'я), `surname` (прізвище), `weight` (вага), `height` (ріст).

### 2. `Car`
Описує технічні характеристики автомобілів.
* Поля: `brand` (марка), `model` (модель), `horsepower` (потужність к.с.), `engineVolume` (об'єм двигуна), `year` (рік випуску), `price` (ціна), `isElectric` (чи є електромобілем).

### 3. `Main`
Головний клас, що містить точку входу в програму (`public static void main`) та виконує всі демонстраційні обчислення.
