# Лабораторна №1

Цей проєкт є консольною програмою на мові Java, яка демонструє базові принципи роботи зі змінними, текстовими рядками, масивами об'єктів та простими математичними алгоритмами.

## 📌 Функціонал програми

Проєкт складається з 5 основних логічних блоків, які виконуються послідовно у методі `main`:

1. **Арифметика змінних:** Розрахунок та виведення суми 10 цілих чисел.
2. **Робота з рядками:** Динамічне об'єднання масиву слів у повноцінне речення за допомогою `String.join()`.
3. **Облік користувачів:** Створення масиву об'єктів класу `User` та обчислення їхнього сумарного віку.
4. **Автопарк:** Створення масиву об'єктів класу `Car` (спортивні, класичні та електромобілі) з розрахунком їхньої загальної потужності в кінських силах.
5. **Математичний реверс:** Алгоритм розкладання тризначного числа на розряди та виведення його у зворотному порядку (дзеркально).

## 🛠 Структура класів

Проєкт містить три основні класи:

### 1. `User`
Зберігає персональні дані користувачів.
* Поля: `id`, `age` (вік), `name` (ім'я), `surname` (прізвище), `weight` (вага), `height` (ріст).

### 2. `Car`
Описує технічні характеристики автомобілів.
* Поля: `brand` (марка), `model` (модель), `horsepower` (потужність к.с.), `engineVolume` (об'єм двигуна), `year` (рік випуску), `price` (ціна), `isElectric` (чи є електромобілем).

### 3. `Main`
Головний клас, що містить точку входу в програму (`public static void main`) та виконує всі демонстраційні обчислення.

# Java Interactive Tasks Console Application

Ця консольна програма на мові Java є збірником інтерактивних завдань, що демонструють роботу з розгалуженнями (`if-else`), оператором вибору (`switch-case`) та введенням даних від користувача через `Scanner`.

Програма пропонує інтерактивне меню, де користувач може вибрати одне з 5 завдань для виконання.

---

## 📌 Опис завдань

Після запуску програма очікує введення номера завдання (**1–5**):

### 1. Квадратне рівняння
* **Що робить:** Обчислює корені квадратного рівняння виду $ax^2 + bx + c = 0$.
* **Вхідні дані:** Коефіцієнти `a`, `b`, `c`.
* **Логіка:** Розраховує дискримінант ($D = b^2 - 4ac$) і виводить два корені, один корінь або повідомлення про їхню відсутність.

### 2. Визначення координатної чверті (Квадрант)
* **Що робить:** Визначає, у якій чверті декартової системи координат знаходиться точка.
* **Вхідні дані:** Координати точки `x` та `y`.
* **Логіка:** Перевіряє знаки координат. Якщо точка лежить на одній з осей, виводить "На осі".

### 3. Перевірка на двозначне парне число
* **Що робить:** Аналізує введене ціле число за кількома критеріями.
* **Вхідні дані:** Будь-яке ціле число.
* **Логіка:** Перевіряє, чи є модуль числа двозначним (від 10 до 99) і чи є воно парним. Виводить "Так" або "Ні".

### 4. Обчислення шматково-заданої функції $f(x)$
* **Що робить:** Розраховує значення функції залежно від діапазону, в який потрапляє аргумент $x$.
* **Вхідні дані:** Число `x`.
* **Логіка:** 
  $$f(x) = \begin{cases} x^2 + 3, & \text{якщо } x < 0 \\ 6\sqrt{x}, & \text{якщо } 0 \le x \le 5 \\ -x + 9, & \text{якщо } x > 5 \end{cases}$$

### 5. Визначення типу дня тижня
* **Що робить:** Визначає, чи є день робочим чи вихідним.
* **Вхідні дані:** Номер дня від 1 до 7.
* **Логіка:** Використовує покращений `switch` (Java 14+). Дні 1–5 — "Робочий", 6–7 — "Вихідний". Будь-яке інше число — "Помилка".
